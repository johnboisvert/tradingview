# Project Summary
CryptoIA Trading Platform is an educational tool that empowers users with knowledge about cryptocurrency trading. It combines analytical tools and resources to assist users in navigating market trends, managing portfolios, and accessing real-time data. The platform features a Python/FastAPI backend and a React/TypeScript frontend, catering to both novice and experienced traders. Recent updates include the integration of NOWPayments for cryptocurrency transactions alongside existing payment options, enhancing user experience and subscription management.

# Project Module Description
The project is structured into several functional modules:
- **Backend**: Manages API requests and data processing using Python and FastAPI, now including support for NOWPayments.
- **Frontend**: An interactive user interface built with React and TypeScript, providing a seamless user experience and updated payment methods.
- **Database**: Utilizes SQLite to store user data, trading history, and application-related information.
- **Deployment**: Uses Railway for cloud hosting and automated deployments.
- **Payments**: Integrated payment options including Stripe, Interac e-Transfer, and NOWPayments for cryptocurrency transactions.

# File Description Inventory
- **Dockerfile**: Defines the environment for the application using Python 3.11 and uvicorn.
- **Procfile**: Specifies the command to start the FastAPI application.
- **requirements-minimal.txt**: Lists essential Python packages needed for the application, including `httpx`.
- **railway.json**: Configuration file for Railway deployment.
- **README.md**: Provides an overview and instructions for the project.
- **lib/store.ts**: LocalStorage persistence service for user data and application state.
- **lib/api.ts**: API service for managing users, messages, ebooks, and plan pricing.
- **pages/Contact.tsx**: Contact form page for user inquiries.
- **pages/Abonnements.tsx**: Updated subscription plans page displaying dynamic pricing, payment options, and NOWPayments integration.
- **pages/Messages.tsx**: Admin page for managing messages and ebooks.
- **pages/Telechargement.tsx**: Page for downloading ebooks.
- **pages/AdminDashboard.tsx**: Admin dashboard displaying key statistics and recent activity.
- **pages/MonCompte.tsx**: User account management page, now includes admin status and plan management.
- **pages/PaymentSuccess.tsx**: Updated page to confirm the success of payment transactions, now supporting both Stripe and NOWPayments.

# Usage
To set up the project, follow these steps:
1. **Clone the repository**: `git clone <repository-url>`
2. **Navigate to the project directory**: `cd <project-directory>`
3. **Install dependencies**: Run `pip install -r requirements-minimal.txt` to install Python packages.
4. **Build the application**: Build the Docker image using the `Dockerfile`.
5. **Run the application**: Use the command specified in the `Procfile` to start the FastAPI server.
